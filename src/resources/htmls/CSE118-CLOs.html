Course CSE118 Learning Outcomes

  1. Articulate how computers work, and the often used numbering systems and their conversions
  2. Apply secure coding techniques to object-oriented programming solutions
  3. Apply the program development process to problems that are solved using fundamental programming constructs and predefined data structures
  4. Choose professional behavior in response to ethical issues inherent in computing
  5. Compare and contrast the primitive data types of a programming language; describe how each is stored in memory; and identify the criteria for selection
  6. Decompose a program into subtasks and use parameter passing to exchange information between the subparts
  7. Describe the language translation phases of compiling, interpreting, linking and executing, and differentiate the error conditions associated with each phase
  8. Differentiate between the object-oriented, structured, and functional programming methodologies
  9. Produce algorithms for solving simple problems and trace the execution of computer programs
  10. Use a professional IDE such as Eclipse or Netbeans for coding and debugging
  11. Write secure programs of at least 100 lines of code with multiple functions and multiple classes involving basic I/O, selections, loops, and arrays
