• Chapter 06: Methods - Learning Objectives

  • To understand method and its usage in Java programming language (CLO6, CLO3)
  • To declare and define a method (CLO6)
    • - To define methods with formal parameters
    • - To define methods with a return value
    • - To define methods without a return value
  • To understand passing arguments by value or by reference (CLO6)
  • To invoke methods with actual parameters/arguments (CLO6)
  • To use method overloading and understand ambiguous overloading (CLO6, CLO4)
  • To determine the scope of variables (CLO5)
  • To apply the concept of method abstraction in software development (CLO6, CLO9)
  • To design and implement methods using stepwise refinement (CLO6, CLO9)
  • To develop reusable code that is modular, easy to read, easy to debug, and easy to maintain (CLO6, CLO9)
